1. Создать класс Main, который запускает наш магазин(public static void main).
2. Создаем класс Shop, у которого должно быть название.Также он должен
содержать Adress, состоящий из индекса, города, улицы, номера дома.
3. В магазине могут быть Goods, имеющие номер, название, тип, цену.
4. В магазине есть список всех товаров.
5. В магазине есть склад, который содержит список товаров, которые есть.
6. Магазин может принимать и списывать товары.
7. Магазин может по чеку продавать покупателю товары.
Если товаров на складе нет, то при добавление товара в чек - должна генерироваться ошибка.
8. Должен быть реализован поиск товара по названию миним из 3 букв.
9. Сделать класс User, который имеет или не имеет право admin(Boolean)
10. Полное Api магазина:admin
- добавить/удалить новый товар в каталог
- принять/списать товар со склада
любой пользователь:
- создать чек со списком товаров и наполнить его товарами с количеством,
(чек должен выдавать общую сумму, после вызова метода продать чек
становится не изменяемым и списывает товары со склада)
- поиск товара в магазине - выдавать название, цены и сумму.
- получить список товаров(сортировать по названию, цене на выбор пользователя)
- получить список товаров наличии(сортировать по названию, цене,
количеству на выбор пользователя)
